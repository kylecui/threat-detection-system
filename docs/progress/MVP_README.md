## ⚠️ 关键风险

### 高风险项 (🔴 Critical)

1. **kSwitch蜜罐失败** (可能性40%)
   - 缓解: Week 4 Go/No-Go决策
   - 降级: L2容器蜜罐

2. **ML数据标注成本** (可能性30%)
   - 缓解: Week 18 Go/No-Go决策
   - 降级: 规则白名单优化

3. **生产切换失败** (可能性10%)
   - 缓解: 双系统并行，灰度发布
   - 降级: 回滚到旧系统

### 中风险项 (🟡 High)

4. **APT验证失败** (可能性50%)
   - 缓解: 简化为3阶段
   - 降级: 持续性评分

5. **Flink性能瓶颈** (可能性30%)
   - 缓解: 提前压测
   - 降级: 简化窗口
