## 🎯 里程碑检查点

### Milestone 1: MVP核心完成 (Week 8)

**验收标准**:
- [ ] 新系统完全替代旧系统
- [ ] 3层时间窗口运行正常
- [ ] 端到端延迟 < 4分钟
- [ ] 蜜罐系统部署 (L1或L2)
- [ ] 规则白名单减少40%误报

**预期效果**:
- 检测准确率提升 +30%
- 误报率下降 -40%
- APT覆盖率 37-64% (取决于蜜罐类型)

---

### Milestone 2: 生产就绪 (Week 12)

**验收标准**:
- [ ] 性能测试通过 (10000 events/s)
- [ ] 监控告警完整
- [ ] 生产环境稳定运行30天
- [ ] 无P0/P1故障

**预期效果**:
- 系统可用性 > 99.5%
- Checkpoint时间 < 30s
- 端到端延迟 < 4分钟 (p99)

---

### Milestone 3: 完整交付 (Week 16-28, 可选)

**验收标准** (取决于决策#2, #3):
- [ ] ML模型部署 (如果决策#2=Go)
- [ ] APT状态机部署 (如果决策#3=Go)
- [ ] 客户满意度 ≥ 80%

**预期效果**:
- 误报率下降 -40% 到 -70% (取决于ML)
- APT检出率 70% 到 95% (取决于状态机)
