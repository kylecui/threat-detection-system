## 🎯 项目目标

将一个传统的基于 C#/Windows/MySQL 的威胁检测系统迁移到**云原生微服务架构**，实现：

1. **跨平台支持**: Windows → Linux (Ubuntu 24.04 LTS)
2. **架构现代化**: 客户端-服务器 → 微服务 + 流处理
3. **性能提升**: 批处理 (10-30分钟延迟) → 实时流处理 (< 4分钟延迟)
4. **可扩展性**: 单机 MySQL → 分布式 Kafka + Flink
5. **功能增强**: 在保持与原系统对齐的基础上增加新检测维度
