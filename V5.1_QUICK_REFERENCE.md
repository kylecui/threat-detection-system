# V5.1 攻击速率权重 - 快速参考卡

## 🎯 核心概念

**攻击速率** = `attackCount / timeWindowSeconds` (次/秒)

## 📊 速率分级表

| 攻击速率 | 权重 | 特征 | 案例 |
|---------|------|------|------|
| **>10/s** | 2.5x | ⚡ DDoS级爆发 | 100次/10秒 |
| **1-10/s** | 1.5-2.0x | 🔥 爆发性攻击 | 30次/30秒 |
| **0.1-1/s** | 1.0-1.5x | 📊 标准扫描 | 30次/5分钟 |
| **0.01-0.1/s** | 0.7-1.0x | 🐌 慢速扫描 | 30次/15分钟 |
| **<0.01/s** | 0.5-0.7x | 🦥 超慢速 | 30次/1小时 |

## 🧪 测试结果 (30次攻击)

```
时间窗口    速率        评分     等级
────────────────────────────────────
30秒        1.0/s       83.96    CRITICAL ⚡
5分钟       0.1/s       79.56    HIGH
15分钟      0.033/s     76.83    HIGH
1小时       0.008/s     75.16    HIGH

评分提升: 30秒比1小时高 11.7% ✅
```

## 💻 API使用

```bash
# 检测爆发性攻击
curl -X POST http://localhost:8083/api/v1/assessment/evaluate \
  -H "Content-Type: application/json" \
  -d '{
    "customer_id": "customer_a",
    "attack_mac": "04:42:1a:8e:e3:65",
    "attack_count": 30,
    "unique_ips": 2,
    "unique_ports": 2,
    "unique_devices": 1,
    "time_window_seconds": 30  ⚡ 关键参数
  }'
```

## 📐 公式变化

**旧版 (V5.0)**:
```
威胁评分 = 基础分 × 时间权重 × IP权重 × 端口权重 
         × 设备权重 × IP段权重
```

**新版 (V5.1)**:
```
威胁评分 = 基础分 × 时间权重 × IP权重 × 端口权重 
         × 设备权重 × IP段权重 × 攻击速率权重 ✨
```
